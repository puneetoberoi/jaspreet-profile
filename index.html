SEARCH ENGINE

This project is divided in 4 Api's with front-end made with Nodejs and Back-end made with Java.

First Node Api has a User interface for the users to input there url into mysql database. This table has a unique property set.

Second Java Api grabs the links from the database with unique tables and grabs all the sub links mentioned on that URL. It checks if the URL exists or no and stores into another mysql. This table also has a unqiue property set.

Third Java Api takes all the sub links, and stores an algorithm result in mongodb database. This database now includes word against links as a key value pair.

Fourth Node Api takes in the user query from the UI and process that query to give results. Here interaction is with the mongodb database which holds key as the word and value as the list of URLs.


WEATHER-API

This project has a single Api which takes in user input and show the weather information.

This site was created by Nodejs and express. It uses data from mapbox.com and darksky.net!

Here we take the user query and talk with two providers. One provides Latitude and Longitude based on the user String. Then we make use of the data above to provide the user with the result.

The Api provides current temperature as well as high and low for the day. It also provides the weather condition with a chance of rain in percentage. Example of the output is as follows -:

Vancouver, British Columbia, Canada Possible light rain throughout the day.It is currently 37.88 This high today is 40.96 with a low of 33.45. There is a 0% chance of rain.


BANK-API

This project can provide the Banking details for new users.

Here we use a utility package where we use List as Data structure. This utility reads the csv file provided which has the username, SIN number and initial deposit information.

We store the information and based on that accounts are created. A new account number is provided and a new pin is created for each user.

Every user gets a public Api to deposit, withdraw and show his information.

It makes use of OOPs concept by making Accounts class and use Interface which defines the default interest rate for the type of account. There are two type of accounts - Checking Account Savings Account
